Projects

Free-for-all Fighting Game

Wrote and designed a free-for-all fighting game using Lua 2.0. Models and meshes were created in RBLX Studio and Blender. Players are randomly placed across the map and must duel until one remains, where money will be given to the winner which they can spend on customising their looks and weapons. Features include an option to spectate other players, custom shop GUI, regional music system, and original map designs.

Maze Exploration Game

Collaborated in a team of 3 to create a small maze exploration game in C++ using the SFML library. The user has to reach an objective somewhere in a randomly generated maze and must avoid enemies. The game uses a special set of algorithms to generate mazes and spawn enemies that can roam around the map.

Text-based Minesweeper

Created a text-based version of Minesweeper using MATLAB. User can choose the difficulty, which will determine the size of the board, and mines will be randomly placed across the grid. Revealing all empty cells results in a win, revealing a mine results in a loss. The game uses optimal calculation methods to quickly reveal cells and show how many mines are adjacent to each cell.
